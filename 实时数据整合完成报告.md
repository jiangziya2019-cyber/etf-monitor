## 五、估值数据改进

### 原方案

```
行业中枢经验值
PE 分位：固定值
股息率：真实数据
准确度：⭐⭐⭐
```

### 新方案

```
实时价格 + 真实股息率
PE 分位：动态计算（实时价格/估算 EPS）
股息率：真实数据（季度更新）
准确度：⭐⭐⭐⭐
```

### 估值计算

```python
# 实时 PE
realtime_pe = realtime_price / eps

# PE 分位（基于历史）
pe_percentile = calculate_percentile(realtime_pe, historical_pe)

# 估值信号
if pe_percentile < 30:
    score += 0.8  # 低估加分
elif pe_percentile > 70:
    score -= 0.5  # 高估减分
```
