```
              行业前景 (1-5 分)

    仓位风险 ←--- 综合判断 ---→ 置信度
```

### 轮动信号检测

**四维评估**：
1. **估值差** (25%) - PE 分位数差异
2. **动量切换** (35%) - 20 日收益率差异
3. **成交量** (20%) - 资金流向确认
4. **趋势** (20%) - 上涨/下跌/震荡

**置信度计算**：
```
confidence = 估值分*0.25 + 动量分*0.35 + 成交量分*0.20 + 趋势分*0.20
```

**信号阈值**：
- 高置信度：≥0.75
- 中置信度：≥0.60
- 低置信度：≥0.45 (放宽阈值)

### 测试验证

**检测到的信号**：
```
【信号 1】新能源 → 高端制造
  置信度：高 (0.76)
  估值差：6.0
  动量切换：10.7%
  逻辑：动量切换；放量确认
  建议关注：机床 ETF (159663)
```

**解读**：
- 新能源行业：PE 40x，20 日收益 -5.2% (下跌趋势)
- 高端制造：PE 34x，20 日收益 +5.5% (上涨趋势)
- 动量切换明显 (+10.7%)，成交量放大确认
- **建议**: 从光伏/储能轮动到机床/卫星
