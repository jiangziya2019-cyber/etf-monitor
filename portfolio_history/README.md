# 持仓历史管理

## 工作流程

### 📅 每日流程

**老板发送**：收盘后发送最新持仓表格
**助手执行**：
1. 读取持仓数据
2. 保存到 `portfolio_history/YYYY-MM-DD.md`
3. 更新 `portfolio_holdings.md`（当前持仓）
4. 计算当日盈亏变化
5. 发送飞书确认

---

### 📊 周一流程（每周复盘）

**时间**：每周一 8:00
**内容**：
- 周末持仓 vs 上周五持仓对比
- 本周操作计划建议
- 持仓盈亏变化分析

---

### 📈 每月 1 号流程（月度复盘）

**时间**：每月 1 号 8:00
**内容**：
- 上月持仓 vs 本月持仓对比
- 月度收益率计算
- 持仓结构变化分析
- 下月投资建议

---

## 文件结构

```
portfolio_history/
├── README.md              # 本文件
├── daily/                 # 每日持仓快照
│   ├── 2026-03-27.md
│   └── ...
├── weekly/                # 每周复盘
│   ├── 2026-W14.md
│   └── ...
└── monthly/               # 每月复盘
    ├── 2026-03.md
    └── ...
```

---

## 数据格式

每日持仓文件包含：
- 持仓明细（代码、名称、数量、成本、现价、盈亏）
- 汇总数据（总市值、总盈亏、仓位分布）
- 调仓记录（新增、清仓、加减仓）
