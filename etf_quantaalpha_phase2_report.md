## 二、数据源详解

### 2.1 行情数据（`data_fetcher.py`）

**双数据源智能切换**：

```
akshare (优先)
  ├─ 主接口：fund_etf_spot_em
  ├─ 备用接口：fund_etf_hist_sina
  └─ 失败 → 切换 Tushare

Tushare Pro (备份)
  ├─ 接口：fund_daily
  ├─ 今日数据无 → 尝试昨日数据
  └─ 都失败 → 使用过期缓存
```

**缓存策略**：
- 盘中（9:00-15:00）：5 分钟过期
- 盘后：30 分钟过期
- 网络波动：自动重试 3 次

**估值分位估算方法**：

```python
# PE TTM 分位（A 股经验法则）
PE < 8   → 10% 分位（极低估）
PE 8-12  → 20-30% 分位（低估）
PE 12-16 → 40-50% 分位（合理）
PE 16-20 → 60-70% 分位（高估）
PE > 20  → 80-90% 分位（极高估）
```
