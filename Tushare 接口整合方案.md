## 三、具体整合方案

### 1. ETF 份额规模监控

**功能**:
```python
# 获取 ETF 份额变化
df = pro.etf_share(ts_code='510300.SH')

# 计算份额变化率
share_change = (latest_shares - prev_shares) / prev_shares

# 判断资金流向
if share_change > 0.05:  # 大幅申购
    signal = '资金大幅流入'
elif share_change < -0.05:  # 大幅赎回
    signal = '资金大幅流出'
```

**应用场景**:
- 每日监控 32 只 ETF 份额变化
- 预警大幅申购/赎回（>5%）
- 辅助调仓决策

---

### 2. 指数实时数据

**功能**:
```python
# 获取指数实时日线
df = pro.rt_idx_k(ts_code='000300.SH')

# 计算 ETF 溢价率
etf_price = get_etf_price('510300')
index_nav = df['close'].iloc[0]
premium = (etf_price - index_nav) / index_nav

# 溢价预警
if premium > 0.05:
    alert('溢价过高')
```

**应用场景**:
- 实时监控 ETF 溢价率
- QDII ETF 溢价监控
- 套利机会识别

---

### 3. 指数分钟数据

**功能**:
```python
# 获取指数 5 分钟数据
df = pro.rt_idx_min(ts_code='000300.SH', freq='5MIN')

# 计算指数动量
momentum = (close - open) / open

# 大盘状态判断
if momentum > 0.01:
    market_state = '强势'
elif momentum < -0.01:
    market_state = '弱势'
```

**应用场景**:
- 大盘实时状态判断
- 第三层策略信号增强
- 调仓时机选择

---

### 4. 技术指标集成

```python
# 获取指数技术指标
```

**应用场景**:

### 5. 全球指数监控

**功能**:
```python
# 获取全球指数
df = pro.index_global(ts_code='DJI/SPX/IXIC/HSI')

overnight_change = (close - pre_close) / pre_close

# 对 A 股影响判断
```

**应用场景**:
