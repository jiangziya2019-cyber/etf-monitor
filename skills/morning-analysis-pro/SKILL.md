## 🧮 综合评分逻辑

```python
综合得分 = 美股得分 (40%) + 期货得分 (20%) + 
           行业得分 (25%) + 概念得分 (15%)

得分 >= +3: 🔥 强烈看好 → 积极补仓，仓位 90-95%
得分 +1~+3: 📈 偏乐观 → 正常补仓，仓位 85-90%
得分 -1~+1: ➖ 中性震荡 → 按计划执行，仓位 80-85%
得分 -3~-1: 📉 偏谨慎 → 减缓补仓，仓位 75-80%
得分 <= -3: ❄️ 谨慎观望 → 暂停补仓，仓位<70%
```
