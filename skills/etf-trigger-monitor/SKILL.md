## 📋 监控逻辑

### 1️⃣ 网格交易触发

**触发条件**:
```python
if current_price <= grid_buy_price:
    # 网格买入触发
    分析行业前景 + 仓位情况
    给出建议：✅ 建议执行 / 🟡 部分执行 / ⚠️ 暂缓 / ❌ 不建议
```

**监控 ETF**: 持仓池 5-8 只 + 观察池 15-20 只

---

### 2️⃣ 止盈触发

**触发条件**:
```python
if current_price >= target_profit_price:
    # 止盈触发
    分析行业前景 + 盈利情况
    给出建议：✅ 建议止盈 / 🟡 部分止盈 / ⚠️ 暂缓
```

---

### 3️⃣ 止损触发

**触发条件**:
```python
if current_price <= stop_loss_price:
    # 止损触发
    分析行业前景 + 亏损情况
    给出建议：✅ 建议止损 / 🟡 观望 / ⚠️ 补仓
```
