## 📊 技能概述

本技能提供完整的四层仓位智能分配能力，基于**融合筛选结果**，将资金分为**防守层/未来层/获利层/全球层**，实现风险分散和收益优化。

**核心能力**:
- ✅ 四层架构智能分配
- ✅ 政策导向因子（未来层 25% 权重）
- ✅ ETF 池扩展（24 只主流 ETF）
- ✅ 动态仓位调整
- ✅ 风险分散（单只权重 2-4%）
- ✅ 自动飞书推送

## 📋 四层架构详解

### 1️⃣ Layer1 防守层（27%）🛡️

**定位**: 高股息 + 低波动，稳定收益

| 属性 | 说明 |
|------|------|
| **风险等级** | 保守 |
| **预期收益** | 8-12%/年 |
| **最大回撤** | <-8% |
| **调仓频率** | 周频 |
| **止损线** | -8% |
| **持有数量** | 6-8 只 |
| **单只权重** | 3-4% |

**优选 ETF**: 红利 ETF、沪深 300、中证 500、上证 180、红利低波、消费 ETF

---

### 2️⃣ Layer2 未来层（27%）🚀 ⭐

**定位**: 政策导向 + 成长型赛道，长期布局

| 属性 | 说明 |
|------|------|
| **风险等级** | 中等 |
| **预期收益** | 15-25%/年 |
| **最大回撤** | <-15% |
| **调仓频率** | 周频 |
| **止损线** | -15% |
| **持有数量** | 8 只 |
| **单只权重** | 2.5-3.5% |

**筛选因子**: 政策导向 25% + 行业景气 20% + 波动率 20% + 动量 20% + 估值 15%

**政策导向 ETF 池** (10 只):
- **硬科技**: 半导体/芯片/人工智能/科创 50
- **新能源**: 光伏/储能电池
- **医药**: 医药创新
- **国家安全**: 军工/数字经济
- **能源安全**: 嘉实原油

---
