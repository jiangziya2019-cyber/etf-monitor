# 持仓记录与收益曲线跟踪机制

## 📋 机制概述

**建立时间**: 2026-03-27  
**触发方式**: 老板每日收盘后发送持仓截图 + 定时复盘提醒

---

## 🔄 工作流程

### 1️⃣ 每日持仓记录（交易日）

**触发**: 老板收盘后通过飞书发送持仓截图

**执行步骤**:
```
1. OCR 识别 → 使用 image-ocr skill 精确识别图片中的持仓数据
2. 整理表格 → 生成完整持仓清单（代码、名称、数量、成本、市价、市值、盈亏、收益率）
3. 发送确认 → 将表格发送给老板核对确认
4. 记录保存 → 保存至 memory/portfolio/YYYY-MM-DD.md
5. 更新曲线 → 更新 memory/portfolio/收益曲线汇总.md
```

**数据原则**:
- ✅ 使用 OCR 工具精确识别，不自己看图推测
- ✅ 无法识别的字段标注"unclear"，不脑补
- ✅ 所有计算用 Python 程序精确执行，不手动估算
- ✅ 重要计算用两种方法交叉验证

---

### 2️⃣ 周度复盘（每周日 11:00）

**触发**: cron 定时任务自动提醒

**执行步骤**:
```
1. 提醒老板发送最新持仓截图
2. OCR 识别并整理持仓表格供确认
3. 更新持仓记录和收益曲线
4. 生成周度复盘报告，包含：
   - 本周市值曲线图表
   - 本周收益曲线图表
   - 持仓变化分析（增减仓、调仓）
   - 盈亏分析（盈利/亏损 TOP5）
   - 行业分布变化
   - 下周展望和建议
5. 发送报告至飞书
```

**cron 配置**:
- 表达式：`0 11 * * 0`（每周日 11:00）
- 时区：Asia/Shanghai

---

### 3️⃣ 月度复盘（每月 1 日 11:00）

**触发**: cron 定时任务自动提醒

**执行步骤**:
```
1. 提醒老板发送最新持仓截图
2. OCR 识别并整理持仓表格供确认
3. 更新持仓记录和收益曲线
4. 生成月度复盘报告，包含：
   - 本月市值曲线图表
   - 本月收益曲线图表
   - 月度盈亏统计
   - 持仓结构变化
   - 行业配置分析
   - 下月策略建议
5. 发送报告至飞书
```

**cron 配置**:
- 表达式：`0 11 1 * *`（每月 1 日 11:00）
- 时区：Asia/Shanghai

## 🎯 关键要求

### 数据准确性
- ❌ 永远不手动估算，必须用程序精确计算
- ❌ 永远不脑补猜测、编造数据
- ✅ 无法识别的字段标注"unclear"或留空
- ✅ 优先使用 OCR 等专业工具精确识别
