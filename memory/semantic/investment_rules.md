# 语义记忆 - 投资规则与模式

## 投资原则（老板底线）

1. **数据真实性**
   - 所有数据必须标注来源和时间
   - 绝不编造数据
   - 行情数据可能延迟 15 分钟，需告知

2. **分析专业性**
   - 分析要有逻辑、有框架、有深度
   - 不说正确的废话
   - 提供分析框架而非买卖时点

3. **沟通态度**
   - 实事求是，不确定的就说不确定
   - 做不到的就说做不到
   - 有风险就说风险

4. **响应机制**
   - 事事有回应
   - 阶段性进展及时汇报
   - 完成后主动报告

## 触发器配置规则

### 五维评分权重
```json
{
  "price": 0.30,
  "volume": 0.25,
  "sector": 0.20,
  "technical": 0.15,
  "market": 0.10
}
```

### 触发阈值
- 综合评分 ≥ 60 分 才触发
- 连续 2 次确认（30 分钟）

### 动态档位
| 市场状态 | 网格档位 |
|----------|----------|
| 牛市 | -3%/-6%/-10% |
| 震荡 | -5%/-10%/-15% |
| 熊市 | -8%/-15%/-25% |

### 仓位管理
| 当前权重 | 加仓比例 |
|----------|----------|
| <5% (轻仓) | 100% (5000 元) |
| 5-8% (正常) | 50% (2500 元) |
| >8% (重仓) | 0% (暂停) |

- 盘后：30 分钟过期

## 调仓规则

### 止损规则
| 行业前景 | 亏损幅度 | 建议 |
|----------|----------|------|
| 好 (4-5 分) | -10% | 暂缓，观察 3-5 天 |
| 好 (4-5 分) | -15% | 部分止损 (50%) |
| 中 (3 分) | -10% | 部分止损 (50%) |
| 差 (1-2 分) | -10% | 全部止损 |

### 止盈规则
| 行业前景 | 盈利幅度 | 建议 |
|----------|----------|------|
| 好 (4-5 分) | <30% | 继续持有 |
| 好 (4-5 分) | ≥30% | 部分止盈 (30-50%) |
| 中 (3 分) | 任意 | 部分止盈 |
| 差 (1-2 分) | 任意 | 全部止盈 |

## 记忆规则

### 必须记录的事项
- 老板说"记住"、"记下来"、"别忘了"
- 重要决策和调仓指令
- 配置变更和系统升级
- 错误和教训

### 记录位置
- **Ontology 图谱**: 结构化实体和关系
- **语义记忆**: 规则、模式、配置
- **情景记忆**: 具体事件和经验
- **每日笔记**: `memory/YYYY-MM-DD.md`

### 禁止
- 不要只口头说"记住了"但不写入文件
- 不要无脑堆砌信息到 MEMORY.md
