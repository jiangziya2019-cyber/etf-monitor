### 2️⃣ 市场状态判断 - 多指数综合判断

#### 修改前
```python
市场状态 = 300ETF 涨跌幅
  > +1% → bull (牛市)
  < -1% → bear (熊市)
  其他 → sideways (震荡)

问题:
  • 单一指数容易被操纵或失真
  • 不能反映全市场真实情况
  • 300ETF 可能受权重股影响
```

#### 修改后
```python
市场状态 = 5 指数加权综合涨跌幅

指数池:
  • 沪深 300 (510300) - 30% 权重 - 大盘蓝筹
  • 中证 500 (510500) - 20% 权重 - 中盘成长
  • 创业板指 (159915) - 20% 权重 - 科技创新
  • 科创 50 (588000) - 15% 权重 - 硬科技
  • 红利指数 (510880) - 15% 权重 - 防御板块

计算公式:
综合涨跌 = Σ(各指数涨跌幅 × 权重)

判断标准:
  • 综合涨跌 > +1.0% → bull (牛市)
  • 综合涨跌 < -1.0% → bear (熊市)
  • 其他 → sideways (震荡)

新增指标:
  • 市场广度 = 上涨指数数 / 总指数数
  • 反映市场涨跌分布情况
```
