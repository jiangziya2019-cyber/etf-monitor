1. **修改 cron 配置**
   ```bash
   crontab -e
   ```

2. **添加/修改任务**
   ```

   0 14 * * 1-5 cd /home/admin/openclaw/workspace && python3 position_adjustment.py
   ```

3. **保存并验证**
   ```bash
   crontab -l  # 确认配置已保存
   ```
