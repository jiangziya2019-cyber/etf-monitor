## 三、完整修复方案

### 方案 A: 整合 akshare 真实历史数据（推荐）

**工作量**: 2-3 小时

```python
import akshare as ak

def get_historical_prices(etf_codes, start_date, end_date):
    """从 akshare 获取真实历史价格"""
    prices = {}
    for code in etf_codes:
        try:
            df = ak.fund_etf_hist_em(
                symbol=code,
                period="daily",
                start_date=start_date,
                end_date=end_date
            )
            prices[code] = df.set_index('日期')['收盘'].to_dict()
        except:
            print(f"⚠️ {code} 数据获取失败")
    return prices

# 在回测中使用
historical_prices = get_historical_prices(etf_pool, '20250101', '20260328')
```
