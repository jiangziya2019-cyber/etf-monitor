## 四、应用场景

### 4.1 资金流向监控

```python
# 监控 ETF 份额变化
etf_share = get_etf_share_size(ALL_32_ETFS, days=5)

# 大幅申购（>5%）：看涨信号
for code, data in etf_share.items():
    if data['signal'] == '大幅申购':
        alert(f"{code}: 资金大幅流入 +{data['share_change_pct']:.2f}%")
```

### 4.2 溢价率监控

```python
# QDII ETF 溢价监控
etf_price = get_etf_realtime_daily(['513110'])['513110']['close']
idx_nav = get_index_realtime_daily(['IXIC'])['IXIC']['close']
premium = (etf_price - idx_nav) / idx_nav

if premium > 0.05:
    alert(f"513110 溢价过高 {premium*100:.2f}%")
```

### 4.3 技术指标分析

```python
# 获取指数技术指标
tech = get_index_techfactor(['000300.SH'])
rsi = tech['000300.SH']['rsi']

if rsi > 70:
    signal = '超买 - 谨慎'
elif rsi < 30:
    signal = '超卖 - 机会'
```

### 4.4 全球市场监控

```python
# 隔夜全球市场表现
global_idx = get_global_index()

# 美股大涨 → A 股高开概率大
if global_idx['SPX']['change_pct'] > 2:
    impact = '正面 - 可能高开'
```

### 4.5 新闻监控

```python
# 重大新闻预警
news = get_news(limit=50)
keywords = ['降准', '降息', '监管']

for n in news:
    if any(k in n['title'] for k in keywords):
        alert(f"重大新闻：{n['title']}")
```
