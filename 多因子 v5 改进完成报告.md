### ✅ 改进 2: Z-Score 标准化

**问题**: Min-Max 归一化对极端值敏感

**解决方案**:
```python
def zscore_normalize(series):
    """Z-Score 标准化 + Winsorization"""
    z_scores = (series - mean) / std
    z_scores = np.clip(z_scores, -3, 3)  # 缩尾处理
    scores = 0.5 + z_scores / 6  # 映射到 0-1
    return np.clip(scores, 0, 1)
```

**应用**:
- ✅ 估值因子使用 Z-Score
- ✅ 资金因子使用 Z-Score
- ✅ 行业中性化使用 Z-Score

**效果**: 评分分布更均匀，对极端值不敏感
