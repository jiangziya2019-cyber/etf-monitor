### 2.2 因子详细设计

#### 因子 1: 估值因子（25%）⭐⭐⭐⭐⭐

**数据来源**: `daily_basic`（真实 PE/PB/股息率）

```python
# 1. PE 分位（历史百分位）
pe_percentile = calculate_percentile(current_pe, historical_pe_3y)
score_pe = 1 - pe_percentile  # 越低越好

# 2. PB 分位
pb_percentile = calculate_percentile(current_pb, historical_pb_3y)
score_pb = 1 - pb_percentile

# 3. 股息率（越高越好）
dividend_score = min(dv_ratio / 6.0, 1.0)  # 6% 为满分

# 估值综合评分
valuation_score = 0.4 * score_pe + 0.3 * score_pb + 0.3 * dividend_score
```

**权重分配**:
- PE 分位：40%
- PB 分位：30%
- 股息率：30%

---

#### 因子 2: 动量因子（25%）⭐⭐⭐⭐⭐

**数据来源**: `fund_daily` / `rt_etf_k`

```python
# 1. 短期动量（5 日）
return_5d = (close / close_5d_ago - 1) * 100
score_mom_5d = normalize(return_5d)

# 2. 中期动量（20 日）
return_20d = (close / close_20d_ago - 1) * 100
score_mom_20d = normalize(return_20d)

# 3. 长期动量（60 日）
return_60d = (close / close_60d_ago - 1) * 100
score_mom_60d = normalize(return_60d)

# 动量综合评分
momentum_score = 0.3 * score_mom_5d + 0.4 * score_mom_20d + 0.3 * score_mom_60d
```

**权重分配**:
- 5 日动量：30%（短期爆发力）
- 20 日动量：40%（中期趋势）
- 60 日动量：30%（长期方向）

#### 因子 3: 波动因子（20%）⭐⭐⭐⭐

**数据来源**: `fund_factor_pro`（60+ 技术指标）

```python
# 1. 20 日波动率（年化）
returns = [close[i]/close[i-1] - 1 for i in range(-20, 0)]
volatility_20d = np.std(returns) * np.sqrt(252)
score_vol = 1 - normalize(volatility_20d)  # 越低越好

# 2. ATR 波动幅度
atr = get_atr(high, low, close, n=20)
atr_ratio = atr / close
score_atr = 1 - normalize(atr_ratio)

# 3. 布林带宽度
boll_width = (boll_upper - boll_lower) / boll_mid
score_boll = 1 - normalize(boll_width)

# 波动综合评分
volatility_score = 0.5 * score_vol + 0.3 * score_atr + 0.2 * score_boll
```

**权重分配**:
- 20 日波动率：50%
- ATR：30%
- 布林带宽度：20%

#### 因子 4: 技术因子（20%）⭐⭐⭐⭐⭐

**数据来源**: `fund_factor_pro`

```python
# 1. RSI 相对强弱
rsi = get_rsi(close, n=14)
if rsi < 30:
    score_rsi = 0.9  # 超卖，买入机会
elif rsi > 70:
    score_rsi = 0.1  # 超买，风险
else:
    score_rsi = 0.5 + (50 - rsi) / 100  # 中性区域

# 2. MACD 趋势
macd, dif, dea = get_macd(close)
if dif > dea and macd > 0:
    score_macd = 0.8  # 金叉 + 多头
elif dif < dea and macd < 0:
    score_macd = 0.2  # 死叉 + 空头
else:
    score_macd = 0.5  # 中性

# 3. KDJ 超买超卖
k, d, j = get_kdj(high, low, close)
if k < 20 and k > d:
    score_kdj = 0.9  # 超卖金叉
elif k > 80 and k < d:
    score_kdj = 0.2  # 超买死叉
else:
    score_kdj = 0.5

# 技术综合评分
tech_score = 0.4 * score_rsi + 0.35 * score_macd + 0.25 * score_kdj
```

**权重分配**:
- RSI：40%
- MACD：35%
- KDJ：25%

```python
# 1. 份额变化率（5 日）
share_change_5d = (shares_5d_ago - shares_now) / shares_5d_ago
score_share = normalize(share_change_5d)  # 份额增长越好

# 2. 成交量变化
vol_change = (vol_today - vol_5d_avg) / vol_5d_avg
score_vol_change = normalize(vol_change)

# 3. MFI 资金流量
mfi = get_mfi(high, low, close, vol, n=14)

    score_mfi = 0.2  # 资金超买
else:
    score_mfi = 0.5

# 资金综合评分
fund_flow_score = 0.5 * score_share + 0.3 * score_vol_change + 0.2 * score_mfi
```

**权重分配**:
- 份额变化：50%
- 成交量变化：30%

### 2.4 市场状态识别增强 ⭐⭐⭐⭐

**新增市场状态判断**:

```python
def identify_market_regime(index_data):
    """识别市场状态"""
    
    # 1. 趋势判断
    ma_20 = get_ma(index_data, 20)
    ma_60 = get_ma(index_data, 60)
    ma_200 = get_ma(index_data, 200)
    
    if index_data[-1] > ma_20 > ma_60 > ma_200:
        trend = '牛市'
    elif index_data[-1] < ma_20 < ma_60 < ma_200:
        trend = '熊市'
    else:
        trend = '震荡市'
    
    # 2. 波动率判断
    volatility = calculate_volatility(index_data)
    if volatility > 0.3:
        vol_state = '高波动'
    elif volatility < 0.15:
        vol_state = '低波动'
    else:
        vol_state = '正常'
    
    # 3. 成交量判断
    vol_ratio = volume_today / volume_20d_avg
    if vol_ratio > 1.5:
        vol_confirm = '放量'
    elif vol_ratio < 0.7:
        vol_confirm = '缩量'
    else:
        vol_confirm = '正常'
    
    return {
        'trend': trend,
        'volatility': vol_state,
        'volume': vol_confirm
    }
```

| **牛市** | 15% | 35% | 15% | 25% | 10% |
| **熊市** | 35% | 15% | 30% | 10% | 10% |
| **震荡市** | 25% | 25% | 20% | 20% | 10% |

### 2.5 筛选流程优化

**新筛选流程**:

```
1. 初筛（流动性过滤）
   ├─ 日均成交 > 5000 万
   ├─ 规模 > 5 亿
   └─ 上市时间 > 6 个月

2. 计算 15+ 因子评分
   ├─ 估值因子（PE/PB/股息率）
   ├─ 动量因子（5/20/60 日）
   ├─ 波动因子（波动率/ATR/布林带）
   ├─ 技术因子（RSI/MACD/KDJ）
   └─ 资金因子（份额/成交量/MFI）

3. 计算综合评分
   └─ 根据市场状态动态调整权重

4. 选择 Top 10-12 只 ETF

5. 风控检查
   ├─ 行业集中度 < 30%
   ├─ 单只 ETF < 10%
   └─ 相关性检查

6. 生成调仓清单
```
